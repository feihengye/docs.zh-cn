# time_to_sec

## 功能

将参数`time` 转换为秒数的时间值，转换公式为:
小时*3600 + 分钟*60+ 秒

## 语法

```Haskell
INT time_to_sec(DATETIME date)
```

## 示例

```plain text
select time_to_sec('12:13:14');
+-----------------------------+
| time_to_sec('12:13:14')     |
+-----------------------------+
|                        43994|
+-----------------------------+
```
